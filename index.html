📥 Import Times from CSV
Upload a CSV with columns: Name, Event, Time, Meet, Date, Gender, Age. Each row adds one time entry.
▶ What format is accepted?
Required columns (case-insensitive, any order):
• Name — swimmer full name
• Event — e.g. Free 50m, Breast 100m, IM 200m
• Time — e.g. 32.45 or 1.12.34

Optional columns:
• Meet — e.g. Intermediate League
• Date — e.g. 2026-03
• Gender — Girls or Boys
• Age — swimmer's age as a number

First row must be a header row. Save from Excel or Google Sheets as .csv.

💡 Tip: The exported CSV from this app can be opened in Excel, edited, and re-imported directly.
